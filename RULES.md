# Global Rules - UNIA

## 🎯 Princípios de Design
1. **Estética Premium**: Interface moderna, vibrante, com micro-animações e foco em UX emocional.
2. **Privacidade": Processamento efêmero para dados sensíveis (conflitos).
3. **Escalabilidade**: Arquitetura baseada em "Nós" para múltiplos tipos de relações.

## 🛠️ Tecnologias
- **Backend**: Node.js/TypeScript (Recomendado pela simplicidade no Docker).
- **Frontend**: Next.js ou Vite + React.
- **Banco de Dados**: PostgreSQL.
- **Deploy**: Easypanel (Docker).

## 📋 Regras de Código
- Verifique sempre `INDEX.md`, `PLAN.md`, `TASKS.md`, `RULES.md` e `WORKFLOW.md` antes de agir.
- RCA (Root Cause Analysis): Sempre investigue a causa raiz (5 porquês).
- DevOps First: Dockerfile e .env robustos.
- Sem modificações não solicitadas.
